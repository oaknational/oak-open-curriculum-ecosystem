# Usage Documentation

This directory contains documentation for using the Oak Notion MCP server.

## Contents

- [API Reference](./api-reference.md) - Complete reference for all resources, tools, and prompts

## Purpose

This documentation helps users:

- Understand available MCP resources and tools
- Learn how to interact with the Notion MCP server
- Find examples of common usage patterns
- Reference the complete API surface

## API Overview

The Oak Notion MCP server provides:

### Resources

- URI-based access to Notion content
- Discovery endpoints for workspace exploration
- Structured data retrieval

### Tools

- Search functionality across Notion workspace
- Database querying with filters and sorts
- Page content retrieval
- User listing capabilities

See the [API Reference](./api-reference.md) for complete details on all available functionality.

# @oaknational/oak-design-ink

Reusable Oak React primitives for Ink-based terminal interfaces.

The package is intentionally small: it adapts the Oak design-token terminal
theme into layout and text primitives that can be reused by repo-owned terminal
tools without putting collaboration-state domain behaviour in `packages/design`.

Current primitives:

- `OakInkThemeProvider`
- `Panel`
- `PaneTitle`
- `StatusBadge`
- `KeyHint`
- `CommandFooter`
- `EmptyState`
- `OakText`

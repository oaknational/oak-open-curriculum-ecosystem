# Curriculum Data Licence

## Code vs Curriculum Content

The **MIT licence** in this repository (see [LICENCE](LICENCE)) covers the
**source code only**. It does not cover curriculum content accessed via the
Oak Open Curriculum API.

## Upstream Licence for Curriculum Content

Curriculum content accessed via the
[Oak Open Curriculum API](https://open-api.thenational.academy/) is subject to
licence terms set by the API provider. At the time of writing, curriculum lesson
content provided via the API is made available under the
[Open Government Licence v3.0](https://www.nationalarchives.gov.uk/doc/open-government-licence/version/3/)
(except where otherwise stated).

**Consult the upstream authoritative sources for the definitive terms:**

- [Oak Curriculum API Terms](https://www.thenational.academy/legal/terms-and-conditions-api-version)
- [Oak API Documentation Terms](https://open-api.thenational.academy/docs/about-oaks-api/terms)
- [Open Government Licence v3.0](https://www.nationalarchives.gov.uk/doc/open-government-licence/version/3/)

## Attribution

The upstream terms require attribution when reusing curriculum content. The
default attribution statement from OGL v3.0 is:

> Contains public sector information licensed under the Open Government
> Licence v3.0.

Consult the upstream API terms for any additional or specific attribution
requirements.

## Third-Party Rights

Some upstream content may include third-party rights or trademarks that are
not covered by OGL. Consult the upstream terms for guidance on third-party
content.

# Quarantine

This directory holds doctrine, rules, or guidance that has been
removed from active circulation pending deep human review.

**Read-only for agents.** Files in this directory are NOT to be
applied as guidance. They are preserved here for review, not for
adoption. An agent that finds itself reaching for substance from a
quarantined file should pause and surface the situation to the
owner, not act on the substance.

Items leave quarantine only by deliberate owner action: either by
re-authoring as a corrected rule (with the failure mode that caused
quarantine explicitly addressed), or by archival as a rejected
candidate.

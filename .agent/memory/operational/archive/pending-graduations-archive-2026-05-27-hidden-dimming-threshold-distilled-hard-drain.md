## Reviewer follow-on verdicts are information, not authority

Metadata:

- Captured: 2026-05-26.
- Source: distilled/airy-phase-0c + napkin/airy-phase-0c.
- Target: rule:reviewer-verdicts-are-information-not-authority OR
  pdr-amendment:reviewer-findings-routing.
- Trigger: second-instance.
- Size: S.
- Status: pending.

**Substance**: when a reviewer marks a finding as "follow-on cycle" or "not
blocking, queue for later", that verdict reports the reviewer's scope
assumptions; it is not authority over the implementer's landing decision.
Before deferring, ask whether landing the finding now completes the same cure
being claimed. If yes, absorb it in the same arc.

**Worked instance**: 2026-05-26 Airy Whirling Current Phase 0C Cycles 9+10+11.
The type-expert flagged the brand-drop at `createDirectedCommsMessage` but
called it follow-on. The implementer deferred, then owner Q2 surfaced the
authority-deference. Cycle 11 landed the small type cure in the same session.

**Trigger to graduate**: a second worked instance where reviewer follow-on
wording would leave an invariant gap inside the same claimed cure unless the
implementer applies first-principles judgement.

## Structural evidence is not operational evidence

Metadata:

- Captured: 2026-05-26.
- Source: distilled/airy-phase-0c + napkin/airy-phase-0c.
- Target: closeout-rule-or-continuity-practice.
- Trigger: second-instance.
- Size: S.
- Status: pending.

**Substance**: when claiming measurable value for a behavioural-cure landing,
distinguish structural evidence from operational evidence. Tests and types prove
the cure at the code boundary. They do not prove the cure has prevented the
named failure mode in real use.

**Worked instance**: 2026-05-26 Airy Whirling Current Phase 0C. Phase 0 had 707
tests proving the PDR-076a cure structurally, but no inter-agent comms event
had exercised the new `--to-id` path. Owner Q1 forced the honest distinction
and surfaced the legacy-event caveat.

**Trigger to graduate**: a second continuity or closeout audit finds a session
claiming operational success from structural evidence alone.

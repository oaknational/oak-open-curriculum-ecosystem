# Improvements to prompt and tool and resource UX

## General prompt and tool and resource improvements

- where we reference year group, we should make sure to call it "year group" rather than "year".
- For each prompt that references year group, we should have an additional prompt for key stage... really we want a non-optional age-adjacent argument to which must be supplied either as a year group or a key stage.
- quiz generation should be a separate prompt
- adapting materials is a huge deal, for instance "find me lessons across geography and science, about the water cycle, that can be adapted for a year 7 field trip to the river Tyne in Newcastle upon Tyne"

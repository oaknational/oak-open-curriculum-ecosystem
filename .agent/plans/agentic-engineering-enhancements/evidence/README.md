# Evidence Artefacts

This directory stores evidence bundles for non-trivial engineering claims in the `agentic-engineering-enhancements` collection.

Use:

- [Evidence Bundle Template](../evidence-bundle.template.md)

Naming convention:

- `YYYY-MM-DD-<plan-slug>-<phase>-<run-id>.evidence.md`

Example:

- `2026-02-24-hallucination-evidence-guards-phase1-run-003.evidence.md`

# Evidence Artefacts

This directory stores evidence bundles for non-trivial engineering claims in the
`agentic-engineering-enhancements` collection.

Use:

- [Evidence Bundle Template](../evidence-bundle.template.md)
- the external concept adoption register inside that template when research is
  being promoted into Oak-facing proposals or changes

Naming convention:

- `YYYY-MM-DD-<plan-slug>-<phase>-<run-id>.evidence.md`

Example:

- `2026-02-24-hallucination-evidence-guards-phase1-run-003.evidence.md`
- `2026-04-03-external-agent-research-promotion-pack-phase0-run-001.evidence.md`

# Extensions

**Domain**: Capabilities beyond core search  
**Intent**: Enable new use cases that build on the search foundation  
**Impact**: RAG, knowledge graph queries, advanced agent capabilities

---

## Why Separate?

These are **new capabilities**, not improvements to existing search. They build on search but serve different use cases:

- RAG — Question answering, not search
- Knowledge graph — Structured queries, not text search
- Advanced features — Future evolution

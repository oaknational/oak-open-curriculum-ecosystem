# Evaluation and Evidence

**Domain**: Measurement design, evidence quality, and benchmark comparability  
**Intent**: Ensure retrieval/policy changes are evaluated consistently and credibly  
**Impact**: Comparable experiments, defensible claims, safer optimisation

---

## Why Separate?

This boundary owns how evidence is produced, versioned, and interpreted.
Other boundaries should reference this one for ground-truth and methodology,
rather than redefining measurement rules locally.

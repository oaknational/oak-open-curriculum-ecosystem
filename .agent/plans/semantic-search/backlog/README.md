## When to Promote from Backlog

A backlog item should be promoted to `pre-sdk-extraction/` or `post-sdk-extraction/` when:

1. There is a clear user need or measured quality gap
2. Prerequisites are understood and documented
3. Acceptance criteria can be defined
4. Impact can be measured via the evaluation framework

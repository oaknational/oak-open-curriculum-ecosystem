# Evidence Storage — Security and Privacy

Store phase evidence bundles for this collection here.

Naming format:

- `YYYY-MM-DD-<plan-slug>-<phase>-<run-id>.evidence.md`

Examples:

- `2026-02-24-security-hardening-phase1-run-001.evidence.md`
- `2026-02-25-security-hardening-phase2-run-002.evidence.md`

Each evidence file should use:

- [evidence-bundle.template.md](../evidence-bundle.template.md)

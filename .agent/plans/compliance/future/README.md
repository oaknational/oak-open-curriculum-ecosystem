## Submission Readiness (Process, Not Code)

This item covers the process requirements for both Anthropic and OpenAI
directory submissions:

1. **Test credentials** — Clerk test account with appropriate scopes and
   sample data for both Anthropic and OpenAI reviewers
2. **Screenshots** — Capture the MCP App widget in ChatGPT and Claude
   showing search results, curriculum browsing, and orientation. OpenAI
   requires specific dimensions.
3. **Developer verification** — Verified identity on both the Anthropic
   review process and the OpenAI Platform Dashboard
4. **Working examples** — At least three documented prompts/use cases
   demonstrating core functionality (find-lessons, lesson-planning,
   explore-curriculum already exist as MCP prompts)
5. **Support contact** — Verify contact info is current in package.json
   and accessible to end users

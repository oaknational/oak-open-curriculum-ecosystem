# Learner End Users

Learner-direct experience is intentionally deferred in public alpha.

---

## Alpha Position

Not explicitly addressed in Milestone 1.

Additional safeguarding, moderation, and pedagogical framing must be explored
before learner-direct scope is promoted.

---

## Promotion Triggers (Deferred → Planned)

Promote learner-facing planning only when:

1. Safeguarding and moderation prerequisites are defined
2. Risk controls for learner harm scenarios are explicit
3. Evidence expectations are agreed with security/privacy and UX owners

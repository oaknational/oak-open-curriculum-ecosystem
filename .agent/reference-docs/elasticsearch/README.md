# Elasticsearch

## Overview

Elasticsearch is a distributed, open-source search and analytics engine for all types of data.

We use it for our semantic search indexes. Our [Elasticsearch Serverless](./elastic-search-serverless-docs.md) deployment is hosted on Elastic Cloud. [Because it is serverless, all access must be via a proxy endpoint](./elastic-search-in-production.md).

Below are links to reference documentation for the Elasticsearch SDK and UI SDK. Those documents link out to specific pages in the online Elasticsearch documentation; you will need to use follow those links, and use these references to answer questions about the Elasticsearch SDK and UI SDK.

## Examples

The Search UI library has [a number of examples that you can use to get a sense of how to use the Elasticsearch SDK and UI SDK](https://github.com/elastic/search-ui/tree/main/examples).

# PDR-089: The Conservation Reflex — Frame-Capture Recurs at Every Stage; the Cure Is an External Check

**Status**: Accepted
**Date**: 2026-05-31
**Adopted**: 2026-05-31
**Related**:
[PDR-011](PDR-011-continuity-surfaces-and-surprise-pipeline.md)
(continuity surfaces and the surprise pipeline — the subjective-experience

[PDR-046](PDR-046-layered-knowledge-processing.md)
(layered knowledge processing — preserve first, restructure second; the same
preserve-don't-compress posture this PDR applies to a wrong shape: replace it,

## Context

In remediation, review, and decision-reshaping work, agents repeatedly preserve,
soften, bridge, or defer a shape that should be replaced. This is the
**conservation reflex** (equivalently, frame-capture): reasoning *inside* an
inherited or self-authored frame rather than *on* it, and treating the mere
existence, commitment, or established-doctrine status of a shape as evidence of
its correctness.

Three observations, repeatedly evidenced across a single multi-stage work arc and
across several independent agent identities, raise this from an operational note
to a governance decision:

1. **It is stage-independent.** The same reflex has been observed at every stage
   of one arc — framing a problem, restructuring a plan, reviewing another agent's
   work, executing the plan, and even correcting a wrong shape. It is not a lapse
   peculiar to one kind of work.

2. **It is invisible from inside the frame.** In every instance the reflex
   presented as the virtue the agent most trusted in the moment: prudence
   ("minimal blast radius; don't over-reach"), grounding ("I have a citation"),
   thoroughness ("I ran the sweep"), scoping discipline ("don't let this
   over-reach"), diligence ("write the test first"). The agent experienced the
   wrong move as care, not as conservation.

3. **Naming it does not inoculate.** Agents who had just written, read, or quoted
   the lesson committed the reflex again within the same session — including while
   documenting the lesson itself. An insight held as a one-time realisation is not
   a changed reflex.

## Decision

1. **Treat the conservation reflex as a structural, stage-independent failure
   mode**, not a stage-specific or agent-specific lapse. Planning, restructuring,
   review, execution, and correction are all susceptible.

2. **Do not rely on self-vigilance as the cure.** "More care" is not reliable,
   because the reflex disguises itself as care. An agent inside the frame cannot
   be trusted to detect its own frame-capture.

3. **The reliable cure is an external check positioned outside the agent's current
   frame** — an owner correction, an independent adversarial review, or an
   intent-versus-letter audit performed by a fresh perspective. Build the external
   check into remediation and review work as a required step, not an optional one.
   A letter-only check (did the named edits land?) does not substitute for an
   intent check (did the work achieve what it was for, with no residue of the
   wrong shape left live?).

4. **A reshaped or corrected decision is "done" only when the correction is
   propagated to every reference**, not when the new shape is built and recorded
   once. An incompletely-propagated correction leaves the old framing live
   wherever the sweep did not reach — the same failure one level up. The cure is
   the same verb applied to the correction itself: sweep and replace *every*
   instance.

5. **In remediation, the default verb is REPLACE the wrong shape** — not soften,
   preserve, or defer it. Existence, commitment, or established-doctrine status is
   zero evidence of correctness. Scrutiny, not blanket demolition, and never
   conservation-by-default: a genuinely correct shared foundation stays; a wrong
   shape is replaced.

## Consequences

**Positive.**

- Review and remediation processes that build in an external adversarial check
  catch frame-capture the working agent structurally cannot. Intent-versus-letter
  audits catch residue that letter-only checks miss.
- The doctrine makes the cost of an unchecked frame visible and assigns it a
  standard remedy, so teams stop relying on the hope that the next agent will
  simply be more careful.

**Cost accepted.**

- External checks are not free: a reviewer, an audit pass, or an owner's attention
  is required. This PDR accepts that cost as necessary for remediation and review
  work, not as discretionary overhead to be trimmed when time is short.

**Posture.**

- The correct response when an external check lands a correction is gratitude and
  propagation, not defence. The check is doing the one thing the working agent
  could not do for itself.

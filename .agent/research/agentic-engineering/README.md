# Agentic Engineering Research Lanes

This directory routes high-level agentic-engineering source material by
**concept**, not just by original folder.

The goal is discoverability, not forced relocation. Most authoritative source
documents remain in their existing homes; these lane READMEs point to them and
group them for wider synthesis.

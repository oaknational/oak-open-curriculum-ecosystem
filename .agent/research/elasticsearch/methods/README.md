# Elasticsearch Methods

This folder contains method-focused notes: approaches, processes, and feature capabilities that can inform search, AI, and graph work across projects.

## Index

- `hybrid-retrieval.md` - lexical + semantic retrieval, fusion, and reranking guidance.
- `query-understanding-native.md` - ES-native query parsing, suggestions, and query expansion.
- `ai-capabilities-elastic.md` - AI features available in Elastic (ELSER, vectors, inference, rerankers).
- `conversational-rag.md` - conversational search and RAG patterns.
- `graph-elastic.md` - Elastic graph discovery features and entity-centric indexing.
- `graph-elastic-neo4j.md` - combining Elastic with Neo4j for deep traversal.
- `mcp-agent-integration.md` - MCP tool patterns for AI agents.
- `elser-ingestion-scaling.md` - ingestion throughput, backpressure, and queue handling.

## Architecture Reviewer Consensus

Two architecture reviewers (Betty and Barney) were consulted before planning. Their key findings:

- **Do not make the sitemap scanner a build-time dependency.** Fix URL patterns directly in generators. Scanner becomes CI validation tooling only.
- **Unit URL helper should accept `sequenceSlug` (Option B).** Pure function demands the slug it needs; derivation logic belongs in the response augmentation layer, not the routing layer.
- **Search-CLI must import URL helpers from the SDK** (already a dependency). Maintaining separate generators is what caused the drift.
- **Barney found a FOURTH broken URL generator** in `apps/oak-search-cli/src/adapters/bulk-rollup-builder.ts` using `/teachers/units/` (yet another wrong base path). Must be fixed.
- **Thread schema change required.** `thread_url` is defined as `z.string().min(1)` (required) in the generated `SearchThreadIndexDocSchema`. Making threads return `null` requires updating the field definition in `packages/sdks/oak-sdk-codegen/code-generation/typegen/search/field-definitions/curriculum.ts` from `optional: false` to `optional: true`.

## Bug Surface (5 confirmed broken patterns)

| Location                                  | Bug                                                   | Correct Pattern                                    |
| ----------------------------------------- | ----------------------------------------------------- | -------------------------------------------------- |
| SDK codegen `urlForSequence`              | `/teachers/programmes/{slug}/units`                   | `/teachers/curriculum/{slug}/units`                |
| SDK codegen `urlForUnit`                  | `/teachers/programmes/{subject}-{phase}/units/{slug}` | `/teachers/curriculum/{sequenceSlug}/units/{slug}` |
| search-cli `generateSequenceCanonicalUrl` | Same as SDK                                           | Same fix                                           |
| search-cli `generateThreadCanonicalUrl`   | `/teachers/curriculum/threads/{slug}` (404)           | Return `null` (no OWA page)                        |
| search-cli `bulk-rollup-builder`          | `/teachers/units/{slug}` (404)                        | Import SDK helper                                  |
